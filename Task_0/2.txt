int - целочисленный тип данных. Выделяется 4 байта

double - тип данных с плавающей точкой. Выделяется 8 байт

char - символьный тип данных. Выделятся 1 байт

float - тип данных с плавающей точкой. Выделяется 4 байта

unsigned int - беззнаковый целочисленный тип данных. Выделяется 4 байта

long long - целочисленный тип данных. Выделяется 8 байт

unsigned long long - беззнаковый целочисленный тип данных. Выделяется 8 байт

bool - логический тип данных. Выделяется 1 байт
